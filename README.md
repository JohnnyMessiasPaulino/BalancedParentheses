# BalancedParentheses
Balanced Parenthesis: Construção de um parser (automato com pilha) - Parser e a segunda etapa de compilação

## Projeto em Java

### Ler um arquivo texto e marcar cada linha com OK ou Inválido:

### {[]} - OK
### ([)] - Inválido
### [{()()}[]] - OK
### {}()[()]] - Inválido
### )[{}]()( - Inválido
### (()[)] - Inválido
### <<[]()>> - Válido
### <<()[<>}> - Inválido
